  This project presents an interactive Electric Vehicle (EV) Analysis Dashboard built using Microsoft Power BI. The dashboard provides insights into EV adoption trends, vehicle types, manufacturers, model years, electric range, and Clean Alternative Fuel Vehicle (CAFV) eligibility.
  Its purpose is to help stakeholders analyze EV market growth, identify leading manufacturers and models, and support data-driven decision-making in sustainability, transportation, and policy planning.
